Project: Doctor prescription
A full-stack application for storing and maintaining patient record in the hospitals & clinics.

FEATURES:
--A complete Form to add the patient data
--update patients data
--search the patient by their name
--delete patient data 
--add patient photo
--delete patient photo from database & from uploads folder
--Show complete data of the patient on browser
--toasts on invalid input from the data
--all screen responsive

TECHNOLOGIES USED:
Frontend:
--React.js
--React Router DOM
--Axios
--React Icons
--CSS
--HTML
--react Toasts

Backend:
--Nodejs 
--Express
--nodemon
--MongoDB
--Multer
--Mongoose
-- express validator

Folder Structure:
Frontend--> contain the all files for the interface of the website
server-->contain the backend logic complete CRUD with Multer

WHAT THIS PROJECT GIVE ME:
--concept of useEffect,useRef,useState
--how to upload picture using multer
--how to make CRUD 
--how to use APIs (get, post,put,delete)
--How to handle git & github
--express validation
